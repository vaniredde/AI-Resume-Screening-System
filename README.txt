# AI Resume Screening System

## Overview

AI Resume Screening System is a Python-based application that analyzes resumes and compares them with a job description using Natural Language Processing (NLP). The system extracts relevant skills, calculates a resume-job match score, and helps recruiters or job seekers quickly evaluate resume compatibility.

---

## Features

- Upload PDF resumes
- Extract resume text
- Detect technical skills
- Compare resumes with job descriptions
- Generate resume match score
- Interactive Streamlit interface

---

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- PyPDF2
- Pandas
- NumPy

---



## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
